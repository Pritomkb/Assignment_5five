Question-2: What is the difference between innerHTML, innerText, and textContent ?

_________________________________________________________________________________________________________

*ইনারএইচটিএমএল, ইনারটেক্সট এবং টেক্সটকন্টের মধ্যে পার্থক্য হলো:
1.এটি একটি DOM (Document Object Model) প্রপার্টি যা একটি HTML এলিমেন্টের ভিতরের HTML কন্টেন্টকে নির্দেশ করে।
2.এটি একটি DOM প্রপার্টি যা একটি HTML এলিমেন্টের ভিতরের টেক্সট কন্টেন্টকে নির্দেশ করে, কিন্তু এটি HTML ট্যাগগুলোকে উপেক্ষা করে।
3.এটি একটি DOM প্রপার্টি যা একটি HTML এলিমেন্টের সমস্ত টেক্সট কন্টেন্টকে নির্দেশ করে, কিন্তু এটি `innerText` এর মতো কিছুটা ভিন্ন।
